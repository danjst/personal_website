Publication: PDAC Heterogeneity via Splicing

Status: Complete (Dec 2020)

My Nature publication that found intron retention events to be strongly linked to survival outcomes in pancreatic adenocarcinoma patients.

PDAC Heterogeneity via Splicing
Technologies used: R, Python, Bash

Changes in AS have previously been linked to the “hallmarks of cancer” i.e. replicative immortality, sustained proliferation, etc which made it a very promising direction in PDAC to investigate. Using techniques such as unsupervised machine learning in conjunction with more specialized analyses like splicing event quantification, I was able to uncover clinically relevant and novel PDAC subtypes distinguished by clear-cut, asymmetric patterns of intron retention in specific groups of genes.

COVID-19 ARDS Time Series Analysis

Status: In-Progress

Survival Analysis and Clinical Outcomes Prediction of COVID-19 ARDS Patients using Real-Time ICU device data.

COVID-19 ARDS Time Series Analysis
Technologies used: R, Bash

In the field of digital health, there has been a growing interest in developing AI- based models capable of incorporating dynamic mechanical ventilation (MV) data to predict various clinical outcomes such as successful weaning in intubated patients. However to the best of our knowledge, no study to date has investigated the use of real-time MV data to predict survival outcomes in COVID-19 ARDS patients. We seek to develop a model capable of learning temporal patterns from combinations of dynamic MV-derived features, e.g., tidal volume, plateau pressure, respiratory compliance, oxygenation, etc., to identify CARDS patients at the highest risk of mortality in real-time. This work can help assist clinicians in identifying which patients are to be prioritized in a fast-paced intensive-care setting. This project is an MTERMS lab project.

Lung Lesion Diagnosis with Computer Vision

Status: Complete (May 2021)

Building a robust deep learning model trained on Stanford's CheXpert Dataset to classify lung lesions from radiographs.

Lung Lesion Diagnosis with Computer Vision
Technologies used: Python (Keras, scikit-learn)

Early identification of lung lesions is critical as lesions have a high likelihood of becoming cancerous. We created an explainable classification model trained on the Stanford CheXpert Dataset and employed various transfer learning and pre-processing techniques to classify lung lesions from radiographs. We found greatest success using the convolutional neural network architechtures VGG16 and DenseNet121. A random forest model was used as a baseline for comparison. After hyperparameter tuning, our final model was able to achieve an AUROC >0.8.

Parallelized t-SNE

Status: Complete (May 2021)

Parallelization of t-SNE algorithm using shared memory parallel architechtures (OpenMP) and accelarated GPU computing (OpenACC).

Parallelized t-SNE
Technologies used: Python, C

t-Distrubuted Stochastic Neighbor Embedding (t-SNE) is a popular dimensionality reduction technique for visualizing large, high-dimensional datasets in low-dimensional space. t-SNE is a highly compute-intensive algorithm involving several expensive matrix operations. In the PCA component of t-SNE, the numerical complexity of the sequential part grows by O(M squared), where M is the number of samples in the matrix while the parallel part grows by O(M cubed). We implemented the t-SNE algorithm in C from scratch that combines concepts from fields such as information theory, numerical optimization and high performance computing. With an implementation in a lower level programming language like C, we open the door to many parallelization and optimization possibilities for t-SNE. In terms of big compute parallelization, we were able to validate our design intentions and tackled areas of the algorithm that yielded the most fruitful speedups.

Adverse Events Interactive Visualization App

Status: Complete (May 2021)

RShiny web app for interactive visualization of FDA Adverse Events dataset.

Adverse Events Interactive Visualization App
Technologies used: R (RShiny, plotly R, tidyverse)

The adverse events dataset is compiled by the U.S. Food and Drug Administration Center for Food Safety and Applied Nutrition (CSFAN). These products include foods, dietary supplements and cosmetics. We designed our visualization primarily for FDA regulatory personnel who are interested in understanding profiles of adverse events caused by in-market products under their jurisdiction. The profiles integrate basic victim, product and medical outcome information to provide regulators a comprehensive view that accounts for all components of an adverse event. Regulators can use this visualization for exploratory analyses and allow these insights to guide regulatory processes and decisions on products under review/in-market.

MetabR

Status: Complete (May 2020)

R package for UCLA Metabolomics quality control and analysis pipeline.

MetabR
Technologies used: R

MetabR is an R package designed for downstream analysis and quality control of LC-MS (Tracefinder) metabolite data. After QC and statistical analysis, the package produces clean and informative visualizations for easy interpretation e.g. identifying significantly upregulated and downregulated metabolites across sample groups/conditions. The package has since been adopted by the UCLA Metabolomics Core facility.
